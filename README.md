# aiteam Landing Page

Landing page giới thiệu aiteam — nền tảng AI tự động build phần mềm thông qua Telegram.

## Ngôn ngữ

Tiếng Việt

## Mục tiêu

- Giới thiệu quy trình AI engineering tự động
- Nhấn mạnh không cần thuê team dev truyền thống
- CTA dẫn user bắt đầu ngay qua Telegram

## Design

- Dark mode
- Accent xanh lam (#3B82F6)
- Phong cách tối giản, hiện đại
- UI dạng SaaS landing page

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Lucide React (icons)

## Cài đặt

```bash
cd frontend
npm install
```

## Chạy development

```bash
npm run dev
```

## Build production

```bash
npm run build
```
