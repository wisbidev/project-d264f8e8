# Test Cases — Hero Section

## Happy Path Scenarios

### TC1 — Headline renders correctly
- [ ] Page displays headline "AI Team. Không cần thuê dev."
- [ ] Headline is visible within the viewport on load (above the fold)

### TC2 — Subheadline renders
- [ ] A subheadline sentence describing the value proposition is visible below the title

### TC3 — Primary CTA button
- [ ] Button with text "Bắt đầu ngay" is visible
- [ ] Button has blue gradient styling (primary appearance)
- [ ] Button is tappable (min 44×44px touch target)

### TC4 — Secondary CTA button
- [ ] Button with text "Xem cách hoạt động" is visible
- [ ] Button has outline styling (secondary appearance)
- [ ] Button is tappable (min 44×44px touch target)

### TC5 — Background styling
- [ ] Section has dark background
- [ ] Section has subtle blue gradient (#3B82F6) overlay

### TC6 — Mobile responsive layout
- [ ] On mobile viewport (≤ 640px), content stacks vertically with no horizontal scroll
- [ ] CTA buttons are full-width or stacked on mobile for easy tapping

### TC7 — Desktop layout
- [ ] On desktop viewport (≥ 1024px), content is full-width and centered
- [ ] CTA buttons are side-by-side horizontally
